# Website Dịch Vụ Sửa Cửa Nhanh

Website landing page cho dịch vụ **sửa cửa sắt – cửa kéo – cửa nhôm tại nhà**.

## 📁 Cấu trúc file

```
website/
├── index.html      # Trang chính
├── style.css       # Stylesheet
├── script.js       # JavaScript (animation, counter)
└── README.md
```

## 🚀 Cách chạy

Mở file `index.html` trực tiếp trên trình duyệt, hoặc dùng Live Server trong VS Code.
